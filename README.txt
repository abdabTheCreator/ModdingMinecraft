# Modding Minecraft

# Opening Intelli
-  Create your own coop-space by using the k8minecraft template on coder.ii
- In the Kitty terminal run: `/opt/idea/bin/idea.sh`

# Running from VSC (version source control) 
- When prompted in the intelliJ windwow choose VSR 
- Paste this git repo link and click continue
